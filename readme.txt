Онлайн кинотеатр из видеоальбомов ВКонтакте.

== Description ==

Импорт видеозаписей из альбомов и со стен групп ВКонтакте.   
Можно выгрузить все видеозаписи с описаниями из группы или со стены, задать автора и рубрику. 
Каждое видео становится отдельным постом WordPress, адаптивный (responsive) плеер ВКонтакте встраивается автоматически.

Cовместим с любой WordPress темой, в том числе с [Free WP Tube 4](http://ukraya.ru/318/free-wp-tube-4 "Бесплатная тема для видео записей").

[PRO версия]( http://ukraya.ru/316/vk-wp-video-pro "VK-WP Video PRO") позволяет установить синхронизацию сайта с видеозаписями на любой стене или из любой группы ВКонтакте. Видеозаписи импортируются на сайт автоматически, в установленное пользователем время. Ряд дополнительных фильтров, позволяет автоматически отсеивать ненужные видео. 

Оперативная [техническая поддержка](http://ukraya.ru/314/vk-wp-video "техническая поддержка") на сайте плагина.

Import videos from video albums and the walls of VKontakte social network.
You can import all the videos and their descriptions, set the author and category. Each video is a separate WordPress post, responsive VKontakte player embedded into post automatically.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Follow the instruction of plugin configuration
